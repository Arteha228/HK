# Зала Богів — Найскладніші боси Hollow Knight

Цей проєкт є інтерактивною галереєю найсильніших супротивників із гри Hollow Knight. Створено в рамках навчання веброзробці (HTML/CSS).

##  Посилання на сайт
**[Подивитися проєкт на GitHub Pages](https://arteha228.github.io/HK/)** 

##  Технології
* **HTML5**: Семантична розмітка, використання метатегів для SEO та OpenGraph.
* **CSS3**: Використання методології BEM, флексбоксів (Flexbox) та адаптивної верстки.
* **Git**: Керування версіями та публікація через GitHub Actions.

##  Особливості проєкту
* **SEO-оптимізація**: Додано опис, ключові слова та метатеги.
* **Доступність**: Для всіх зображень прописані атрибути `alt`.
* **Дизайн**: Темна висококонтрастна тема в стилі гри для кращого сприйняття.
